Week 5 - Bivariate Data, Functions and Linear Models



  • The data need not pass the vertical line test.
  • The model(function) must pass the vertical line test.
  • y as a function of x
  • Similar to y = mx + c
  • linFit(dataset$independentvariable(x-axis), dataset$dependentvariable(y-axis)) This will give intercept value and slope, thereby allowing us to determine y = mx + c equation.
  • There will arise situations in which the generated intercept value would be negative and hence would not be applicable in real life situations. In such cases we must convert our x-axis (independent) variable more usable...i.e. making the minimum value of provided data for the x-axis equal to 0.
  • eg - Pop$new <- Pop$Population - min(Pop$Population)......this will subtract the minimum value from all the x-axis data points, therefore relative to each other, the x-variable data points would still be at the same distance and hence, there would not be any change in the slope or coefficient of determination but it will ensure that the y-variable minimum value remains positive.
  • R^2 (Coefficient of determination)....closer it is to one.....that much proportion of the variance in the dependent variable can be attributed to independent variable. eg - 0.97....means 97% proportion of the variance of the y-axis variable is due to the x-axis variable.
