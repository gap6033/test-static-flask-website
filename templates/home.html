About the Website

A Lawyer's Coding Repository

This website,for the time being, merely serves as an expirement in my journey from being a lawyer to becoming familiar with the computer science universe. I have created it to explore the realm of web devlopment. Currently, I am using the Flask framework for deploying the site and relying on the Bootstrap framework for styling it. I have begun first by developing the front-end; developing an interactive and responsive but static web application that presently is capable of only taking in the "GET" requests from users. However, as I proceed further ahead in my exploration, I do intend to utilize functionality of the "POST" method as well thereby creating a web application that allows users to POST data as well and for which I'll be taking assistance of SQL language. In the meanwhile, in order to better explore the functionality of this website, I'll be using the content (unrelated to web devlopment) from the various online CS courses that I undertook.
