Week 3 - Plotting bivariate data in R



Plotting Bivariate Data in R
  • import dataset
  • plot(datasetname$variable(for x-axis-the independent variable), datasetname$variable (for dependent variable on y-axis), (optional)xlab = '', ylab = '', main = '').This will plot a scatterplot for two QUANTITATIVE variable.
  • For viewing the line through scatterplot - abline(lm(datasetname$variable(dependent one)~datasetname$variable(independent one)))
  • For calculating 'r' - Pearson correlation coefficient cor(datasetname$variable(for x-axis-the independent variable), datasetname$variable (for dependent variable on y-axis))
  • Coefficient of determination = r^2
